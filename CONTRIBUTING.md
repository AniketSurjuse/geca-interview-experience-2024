
# Contributing to GECA Interview Experience 2024 
   
First off, thank you for considering contributing to GECA Interview Experience. It's people like you that make this community such a great resource for our juniors.  
   
## How to contribute?  
1. Fork the repository to your GitHub account.  
2. Clone the forked repository to your local machine.  
3. Make your changes in your local repository.  
4. Commit your changes and push them to your forked repository.  
5. Submit a Pull Request (PR) to the original repository.  
   
## What should I know before I get started?  
Please follow the existing structure for adding your interview experiences. Create a new markdown file under the respective company folder (create a new folder if it doesn't exist already). 
Its okay to use ChatGPT to finetune your experience but dont over use it. if you know what I mean ;)
  
Make sure to include the following information in your markdown file:  
   
1. **Company Name**  
2. **Job Profile**  
3. **Interview Date**  
4. **Interview Rounds Details**  
   - Description of each interview round  
5. **Questions Asked**  
   - Technical and HR questions  
6. **Tips and Suggestions**  
7. **Your Overall Experience**  
   
Please name the markdown file in the following format: `CompanyName_YourName.md`  
   
## How can I report a problem or suggest a new feature?  
Please open a new issue in the repository for any problems you encounter or enhancements you suggest.  
   
## Code of Conduct  
We are committed to fostering a welcoming community. Any form of offensive behavior, harassment, or bullying will not be tolerated.   
  
Thank you again for your interest in contributing. Happy sharing!
